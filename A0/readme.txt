To modify which file we are using and what type of splay, do the following:

In mainsplay.cpp, comment out 3 of the lines 11-14, leaving only the one you want to use uncommented *(This will select the access pattern to use)*
In splay.cpp, comment out 3 of the lines 454-457, leaving only the one you want to use uncommented *(This will select the type of splay to use)*
